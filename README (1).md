## Setup & Run (Windows)

### 1. Install dependencies
```powershell
py -m pip install scikit-learn xgboost shap fastapi uvicorn python-multipart pandas numpy
```

### 2. Train the model
```powershell
py train.py
```

This auto-generates inside `models/`:
- `phishguard_model.pkl` — trained XGBoost classifier
- `shap_explainer.pkl`   — SHAP TreeExplainer
- `feature_names.json`   — feature schema
- `training_report.json` — full metrics

### 3. Start the API
Open a terminal and run (keep it open — do not close):
```powershell
py -m uvicorn api.main:app --reload --port 8000
```

You should see:
```

```

## Integrating Kaggle Datasets

You have two Kaggle datasets: `dataset_phishing.csv` and `phishing.csv`.
To use them instead of the synthetic data, replace the dataset loading in `train.py`:

```python
# At the top of train.py, replace the build_dataset() call with:

def load_kaggle_datasets():
    import pandas as pd
    dfs = []

    # Dataset 1: dataset_phishing.csv (URL-based features, 87 columns)
    df1 = pd.read_csv("data/dataset_phishing.csv")
    df1['label'] = (df1['status'] == 'phishing').astype(int)
    df1 = df1.drop(columns=['status', 'url'], errors='ignore')
    dfs.append(df1)

    # Dataset 2: phishing.csv
    df2 = pd.read_csv("data/phishing.csv")
    for col in ['label', 'status', 'Result', 'class', 'phishing']:
        if col in df2.columns:
            df2['label'] = (df2[col].astype(str).str.lower()
                            .isin(['phishing', '1', '-1', 'yes'])).astype(int)
            df2 = df2.drop(columns=[col], errors='ignore')
            break

    df = pd.concat(dfs, ignore_index=True, sort=False).fillna(0)
    print(f"Kaggle dataset: {len(df)} rows, {df['label'].sum()} phishing")
    return df

# Then replace:
#   df = build_dataset()

#   X = df.drop(columns=['label'], errors='ignore')
#   y = df['label'].values
# (since Kaggle data already has numeric features — no need for feature_extractor.py)
```

## Upgrade Path for Research

### Add BERT embeddings
```python
from transformers import BertTokenizer, BertModel
import torch

tokenizer = BertTokenizer.from_pretrained('bert-base-uncased')
bert = BertModel.from_pretrained('bert-base-uncased')

def get_bert_embedding(text):
    inputs = tokenizer(text, return_tensors='pt', truncation=True, max_length=512)
    with torch.no_grad():
        outputs = bert(**inputs)
    return outputs.last_hidden_state[:, 0, :].numpy()  # [CLS] token
```

### Add LIME for text explanations
```python
from lime.lime_text import LimeTextExplainer

explainer = LimeTextExplainer(class_names=['Legitimate', 'Phishing'])

def explain_with_lime(text):
    exp = explainer.explain_instance(
        text,
        lambda texts: np.array([[1-p, p] for p in [predict_proba(t) for t in texts]]),
        num_features=10
    )
    return exp.as_list()
```

## XAI Methods Used

| Method | What it does |
|--------|-------------|
| **SHAP TreeExplainer** | Per-prediction feature contributions (± impact on score) |
| **XGBoost feature_importances_** | Global feature ranking across all training samples |
| **Highlighted text spans** | Which exact words/phrases triggered the detection |
| **Attack type taxonomy** | Classifies into: credential_harvest, BEC, IT_impersonation, malware, advance_fee |
| **Indicator scores** | Urgency / Deception / Authority / Payload — four independent axes |
